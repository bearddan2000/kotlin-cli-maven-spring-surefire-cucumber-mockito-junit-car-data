# kotlin-cli-maven-spring-surefire-cucumber-mockito-junit-car-data

## Description
A POC for spring app using junit5
and cucumber framework with surefire
plugins.

## Tech stack
- kotlin
- maven
	- mockito
  - spring
  - junit5
  - cucumber
  - surefire

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
